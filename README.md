Задание 1
Опишите основные преимущества использования масштабирования методами:

активный master-сервер и пассивный репликационный slave-сервер;
master-сервер и несколько slave-серверов;
Дайте ответ в свободной форме.

1. Активный master‑сервер и пассивный репликационный slave‑сервер

Основные преимущества:

Высокая доступность. Пассивный slave‑сервер может быстро заменить master в случае сбоя — это сокращает время простоя системы.

Резервное копирование без нагрузки на мастер. Реплика используется для создания бэкапов: операции копирования не затрагивают рабочий сервер.

Разгрузка мастер‑сервера. Slave‑сервер обрабатывает запросы на чтение (SELECT), снижая нагрузку на master, который фокусируется на операциях записи (INSERT, UPDATE, DELETE).

Тестирование и аналитика. На реплике можно запускать тяжёлые аналитические запросы или тестировать обновления без риска для продуктивной среды.

Простота настройки и поддержки. Архитектура master‑slave относительно проста в развёртывании и администрировании по сравнению с более сложными схемами.

2. Master‑сервер и несколько slave‑серверов

Преимущества по сравнению с одним slave:

Улучшенная отказоустойчивость. При выходе из строя одного slave‑сервера нагрузка перераспределяется между оставшимися репликами.

Масштабирование чтения. Запросы на чтение распределяются между несколькими серверами — это повышает общую производительность и пропускную способность системы при высокой нагрузке.

Географическое распределение. Slave‑серверы можно разместить в разных регионах: пользователи получают данные с ближайшего узла, что снижает задержки (latency).

Балансировка нагрузки. Использование балансировщика (например, HAProxy) позволяет равномерно распределять запросы между репликами, избегая перегрузки отдельных серверов.

Гибкость сценариев использования. Разные slave‑серверы могут выполнять специализированные задачи: один — для отчётов, другой — для бэкапов, третий — для тестирования.

Плавное масштабирование. Добавление новых slave‑серверов позволяет наращивать мощность системы по мере роста нагрузки без остановки работы.
